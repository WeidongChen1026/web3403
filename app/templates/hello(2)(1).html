This game is a puzzle game, for users to relieve stress and pass the time in their free time.

Game rules:
1. New users need to register an account, and please remember your account and password
2. After entering the game, the system will automatically generate a random number, the range of the random number is between 1-99, and the user needs to guess within this range.
3. If the guessed number is too large or too small, the system will automatically prompt you, you only need to modify it according to the prompt.
4. There are 3 modes in this game, the easy mode has 10 chances to guess, the hard mode has 5 chances, and the difficult mode has 2 chances.
5. You need to guess the number correctly before the number of times is 0. If you guess correctly, you can add one point. If you guess wrong, no points will be awarded.
6. If you like our game, please play with your friends and see who can guess more times!
